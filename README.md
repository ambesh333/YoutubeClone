# Youtube Clone

A brief description of the project.

## Installation

1. Clone the repository:
```shell
git clone https://github.com/ambesh333/YoutubeClone.git
```
3. Install the dependencies using npm:
```
npm install
```
4.To start the website, run the following command:
```
npm run start
```

